# Changelog

## v0.2.0-alpha

### Highlights
- Local PBS editor with safe export to `PBS_Output/`
- Multi-page editors for core PBS files
- Cross-file references (Types/Abilities/Moves/Items)
- Non-destructive export, validation, warnings
- Encounter/Trainer editors with multi-section handling
- Pokemon + Forms editors with graphics preview

### Supported PBS
- types.txt
- abilities.txt
- berry_plants.txt
- ribbons.txt
- moves.txt
- items.txt
- trainer_types.txt
- encounters.txt
- trainers.txt
- pokemon.txt
- pokemon_forms.txt

### Key Features
- Add/Duplicate/Delete entries
- Move Entry modal with group handling for versions
- Per-field validation with export blocking on errors
- Export formatting matches Essentials conventions
- Icons/sprites support for Pokemon, Forms, Items, Types, etc.

### Known Limitations
- Encounters/Trainers/Forms UX still alpha
- No installer yet (manual zip of release output)
- Essentials 21.0 expansions not implemented
